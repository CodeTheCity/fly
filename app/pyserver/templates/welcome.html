Welcome

This is the back-end API for the FLY project at codethecity

API

GET /time - UNIX time of the next quest start

GET /quest/species/ - list of five notable natural things, at random

POST /find/ - send a JSON document representing a find.

GET /finds/recent - look at recent finds

POST /user/ - send a JSON document representing a user
curl -d 'data={"test":1}' http://hostname/url/

GET /current_quest/ - get the latest quest

POST /quest/new/ - generate a new quest

GET /finds/map/[string] - search for finds
